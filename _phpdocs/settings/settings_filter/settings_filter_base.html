[line 28]

Entity of "filter". In most cases only calling $callback on provided pair references $name/$value in method apply().

But it is quite powerful. Childs of this basic class may provide any service such as: non-deterministic 	state-based results, based on time or amount of call results (f.e. Apply filter only 3 times!?), depend 	on any global environment etc...
