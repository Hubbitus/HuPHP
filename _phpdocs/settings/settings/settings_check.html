method nesting [line 141]

void nesting( )

Emulate nesting.

As we reimplement object to do not have properties itself, instead define it in $this->properties we should provide mechanism to emulate nestiong, to do not mention each time again presented properties. So, with this method we can define in childs new propery $this->properties_addon and than call this method (in constructor f.e.) to add new props.

So, method MUST be called explicitly. No any magic here!!!
