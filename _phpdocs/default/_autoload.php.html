Autoload for MACROSES (and also all functions)!!!

PHP compleatly do not support autoloading of functions, only classes. So, we may do that DIRTY HACK: Instead of just using function like MY_COOL_FUNCTION() precedes it by singleton: m()->MY_COOL_FUNCTION(). In PHP >= 5.3.0 (When introdused magick methos __callStatic) we may use more efficient way for that, like just: m::MY_COOL_FUNCTION(). But it is not supported in previous versions.

Eval used to hide from current version of PHP and avoid errors/warnings.
