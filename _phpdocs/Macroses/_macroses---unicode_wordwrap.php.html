string unicode_wordwrap( string $str, [integer=75 $len = 75], [string=\n $break = &quot;\n&quot;], [boolean=false $cut = false])

wordwrap standard function wrap by amount *bytes*, not *chars*! Idea got from http://ru.php.net/manual/en/function.wordwrap.php#92577 but its implementation don't work. Reimplemented.



Parameters

string   $str   String to wrap
integer=75   $len   Length to wit in.
string=\n   $break   String to place on end
boolean=false   $cut   Cut words or not (Default false, to wrap by word boundary).
